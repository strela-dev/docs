## Getting started {{ anchor: false }}

To get started, create a Kubernetes cluster or use an existing one. Then install Strela and create a new [MinecraftDeployments](/minecraft-deployment) or [MinecraftStatefulSets](/minecraft-stateful-set), and you're ready to go. Additionally, you can create an [MinecraftAutoscaler](/minecraft-autoscaler) to automatically scale your Minecraft server instances, based on the number of players or available servers. {{ className: 'lead' }}
